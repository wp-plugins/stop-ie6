== Description ==

This plugin identifies if a visitor is using Internet Explorer 6 (or older) and suggests to upgrade to a recent browser. 

== Installation ==

1. Upload the folder 'stopIE6' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. This is it!
